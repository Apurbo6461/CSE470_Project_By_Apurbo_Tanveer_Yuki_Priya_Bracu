# LifeLink Frontend

This is a Vite + React frontend for the LifeLink demo.

Setup

1. Install dependencies:

   npm install

2. Set API URL by copying `.env.example` to `.env` (`VITE_API_URL` defaults to http://localhost:5000)

3. Start dev server:

   npm run dev

The UI uses Leaflet for the map and Axios to call the backend API.
